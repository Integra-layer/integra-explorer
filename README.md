# Integra Explorer

Next-gen block explorer for [Integra Chain](https://integralayer.com).

Built on [Ethernal](https://github.com/tryethernal/ethernal) (MIT license).

## Quick Start

See [docs/deployment.md](docs/deployment.md) for deployment instructions.

## Architecture

- **Backend:** Ethernal (Express + TimescaleDB + BullMQ + Soketi)
- **Frontend (Phase 1):** Ethernal Vue 3 (stock)
- **Frontend (Phase 2):** Next.js 16 + shadcn/ui + Tailwind + Framer Motion + Magic UI + Aceternity UI
